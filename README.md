### [+] Description
nmapXLSX converts nmap output in XML format to XLSX format.

### [+] Installation
 - `go install github.com/TURKZEN/nmapXLSX@latest`

### [+] Usage
 - `nmapXLSX <xml_Nmap_Output> <Output> `
